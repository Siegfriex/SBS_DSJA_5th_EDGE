# H1/H2/H3 심화 시각화 보고서

## 핵심 요약

- H1은 여전히 가장 강한 신호다. 순위·상위도와 A비율의 추세선과 Spearman heatmap이 같은 방향을 보인다.
- H2는 시각화에서도 약하다. 학점포기제 O/X의 100% 성적구성 차이는 순위 급간 차이보다 기사적으로 작게 읽힌다.
- H3는 취업성과보다 진학률 쪽 보조 신호가 더 뚜렷하다. A잔차와 공식취업률 패널은 거의 수평에 가깝다.
- 3D PCA는 설명 보조 그림이다. 입력 변수에는 A비율 자체를 넣지 않았고, 색상으로만 표시했다.

## 안전 문장

2024년 대학 서열 프록시 기준으로 상위권 대학의 A비율이 높은 방향은 비교적 일관되게 관찰된다. 반면 학점포기제 O/X만으로는 A비율 차이를 설명하기 어렵고, 서열 대비 A를 적게 주는 대학이 취업성과에서 뚜렷하게 우수하다는 결론도 현재 산출물로는 무리다.
