## 3. 0값 처리 결론

수치형 컬럼의 0은 NULL로 바꾸면 안 된다.

- `입학정원/모집인원/지원자/입학자`: 경쟁형 수치다. 0은 미모집, 미지원, 미입학 가능성이 있는 유효 관측값이다.
- `재적생/외국인유학생/졸업자/교원`: 집계형 수치다. 0명은 실제 0명으로 유지한다.
- 파생 비율은 별도 처리한다. 예를 들어 경쟁률 계산에서 분모가 0이면 원값을 NULL로 바꾸지 않고 계산 결과만 NULL 처리한다.

`시간강사_계/남/여`는 세 컬럼 모두 전 행 0이므로 분석용 `df`에서 제거했다. 원본 Excel에는 남아 있고, 제거 내역은 `tables/22_dropped_analysis_columns.csv`에 별도 기록했다.

## 4. 가장 중요한 발견: `소속학과없음`

`소속학과없음` placeholder 행은 `462행`이다. 이 행들의 `대계열/중계열/소계열` NULL은 실질 결측이 아니라 구조적 NULL이다.

다만 단순히 삭제하면 안 된다. `소속학과없음` 행에는 교원 수치가 배정되어 있다.

| 컬럼 | placeholder 총합 | 비0 행수 | 처리 |
|---|---:|---:|---|
| `비전임교원_계` | 8,234 | 328 | 총합에서는 별도 집계 |
| `비전임교원_남` | 4,752 | 300 | 총합에서는 별도 집계 |
| `비전임교원_여` | 3,482 | 254 | 총합에서는 별도 집계 |
| `전임교원_계` | 459 | 323 | 총합에서는 별도 집계 |
| `전임교원_남` | 384 | 290 | 총합에서는 별도 집계 |
| `전임교원_여` | 75 | 53 | 총합에서는 별도 집계 |

따라서 처리 원칙은 다음이 맞다.

- 학과명/계열별 평균, 분포, 순위 분석: `소속학과없음` 제외
- 학교/시도/전체 교원 총합 분석: `소속학과없음`을 별도 항목으로 유지
- 입학정원, 지원자, 입학자, 재적생, 졸업자 분석: `소속학과없음` 행은 모두 0이므로 포함 여부가 총합에는 영향이 거의 없지만, 평균/분포에는 왜곡을 만든다

## 7. 사용자가 결정해야 할 사안

### 결정 1. 분석 단위

권고: 기본 분석 단위는 `연도 + 학교명 + 본분교 + 주야구분 + 학위과정 + 학과코드`로 둔다.

선택지:

- A. 학과코드 기준: 가장 안정적이다. 같은 학과명이 여러 학교/과정에서 반복되어도 구분 가능하다.
- B. 학과명 기준: 사람이 보기 쉽지만 동명이과 문제가 생긴다.
- C. 대계열/중계열/소계열 기준: 큰 흐름 분석에는 좋지만 세부 학과 분석은 못 한다.

### 결정 2. `소속학과없음` 처리

권고: 이중 처리한다.

- 학과별 평균/분포/순위: 제외
- 학교/시도/전체 교원 총합: 별도 항목으로 유지

무조건 삭제하면 교원 총합이 빠지고, 무조건 포함하면 학과 분포가 왜곡된다.

### 결정 3. `대학원구분` NULL 처리

권고: 분석용 컬럼 `대학원구분_clean`을 쓴다.

- 대학원과정: 원래 값 유지
- 비대학원: `해당없음_비대학원`

원본 NULL 기준으로 행을 제거하면 전체의 64.2%를 잘못 버린다.

### 결정 4. 시간강사 컬럼 처리

처리 완료: `시간강사_계/남/여`는 분석용 `df`에서 제거했다.

- 제거 전 원본 정리 컬럼 수: 129
- 제거 후 분석 컬럼 수: 126
- 제거 후 수치형 컬럼 수: 105
- 제거 컬럼은 기술통계, 실질결측, 0값 audit, placeholder audit에서 제외됨

이 항목은 더 이상 사용자가 결정할 사안이 아니다. 원본 보존 + 분석 제외로 확정했다.

### 결정 5. 경쟁률/충원율 같은 파생변수 생성 방식

권고: 원값 0은 유지하고, 파생변수 계산 결과만 조건부 NULL 처리한다.

예시:

- `지원경쟁률 = 지원자_전체_계 / 모집인원_학부_계`
- 분모가 0이면 `지원경쟁률 = NULL`
- 이때 `지원자_전체_계`나 `모집인원_학부_계` 원값을 NULL로 바꾸지는 않는다
