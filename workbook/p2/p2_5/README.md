## Interpretation Guardrails

- AME values are conditional associations, not causal effects.
- RAW_A and residual grade-signal branches must be read separately.
- Context profiles are descriptive with only seven major groups.
- P2-Q selectivity-blocked artifacts should not be treated as finished model evidence.
