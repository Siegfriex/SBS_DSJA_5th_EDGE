## Engineering Research Value

- Tests whether the grade signal behaves differently by major group and outcome.
- Separates RAW_A strict estimates from residual branches that depend on upstream P3 artifacts.
- Provides AME interpretation in percentage points for graduate-school progression and health employment.
- Adds a visual-insight notebook with lineage, sample contract, model diagnostics, selectivity audit, and P5 heterogeneity heatmaps.
